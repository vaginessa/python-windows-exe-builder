The Easy Executable Builder for Python

Python is a great language.

However, distributing Python apps is hard.

There are loads of hacks needed to get it to build into a portable executable on your current platform, let alone cross-compile.

What does this script do?

Simplify. This distills the process into a simple script that asks you simple questions, and you get an executable.

Warning: Pre-Alpha!

This project is in the pre-alpha stage, but should go stable in a few weeks.
